# **App Name**: Firebase SEO Site Factory

## Core Features:

- Website Creation and Management: Create, delete, and manage multiple websites within the Firebase environment. Each website will be stored as JSON data in Firestore.
- AI-Powered Content Generation: Generate page content using a generative AI tool, for various sections like Home, About, Services, and Contact. Customize the generation based on the site's purpose, target audience, and specific keywords.
- Automated Styling and Layout: Automatically style and layout each page using AI-driven design suggestions and pre-built templates, ensuring a consistent and visually appealing design.
- Location Management for Local SEO: Incorporate location data for local SEO. Fetch and store location information using Google Maps Geocoding API and embed it within the website's schema and content.
- Live Site Preview with Inline Editing: Expose each website under a unique slug (e.g., /site/{slug}) with a live preview. Allow inline editing of content and CSS, with changes persisted to Firestore.
- Admin Dashboard: Provide a Firebase-hosted admin dashboard to manage websites, pages, and settings.
- SEO Audit Integration: Integrate SEO audit tools such as Unlighthouse to assess the website's SEO performance and provide actionable insights. Reports are saved into the Firestore.

## Style Guidelines:

- Primary color: Deep violet (#9400D3) evoking sophistication and technological prowess.
- Background color: Very light grayish-purple (#F0F0F7). This color offers a neutral, desaturated backdrop that enhances the vibrancy of the primary and accent colors while maintaining a professional feel.
- Accent color: Cyan (#00FFFF) used sparingly for interactive elements, call-to-action buttons, and highlights. Its brightness against the darker primary color ensures it grabs attention without being jarring.
- Headline font: 'Space Grotesk' sans-serif for a modern, tech-focused aesthetic.
- Body font: 'Inter' sans-serif providing a clean and readable experience.
- Minimalist and modern icons to represent different website features and functionalities.
- Clean and responsive layout using a grid system for easy content organization.
- Subtle animations and transitions to enhance user experience and engagement.
